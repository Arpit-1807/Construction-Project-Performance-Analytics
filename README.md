## Business Problem

Construction companies often struggle with:
- Tracking project progress
- Monitoring manpower utilization
- Identifying delays
- Controlling operational costs

This dashboard was developed to help management monitor project performance and make data-driven decisions.

## Objectives

- Analyze project-wise financial performance
- Monitor workforce utilization
- Identify delay trends
- Track project profitability
- Detect operational risks

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Excel
- Data Modeling

## Key KPIs

- Total Billing
- Total Cost
- Profit Margin %
- Avg Daily Progress %
- Workforce Utilization %
- Delay %
- Delay Days

## Key Insights

- Metro project achieved the highest profitability despite lower operational cost.
- Bridge project showed the highest financial exposure and workforce demand.
- Projects with higher delays exhibited increased operational expenses.
- Workforce shortages may contribute to project execution delays.

## Business Impact

The dashboard helps stakeholders:
- Monitor operational efficiency
- Detect cost overruns
- Improve manpower planning
- Reduce project delays
- Support data-driven decision-making
